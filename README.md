# Quiz Genie 🎓✨

**Quiz Genie** is an interactive and intelligent quiz platform designed as a final-year project. It allows users to create, manage, and take quizzes efficiently. Designed with both usability and scalability in mind, Quiz Genie is perfect for students, educators, and developers alike.

## 🚀 Features

- 📝 Create custom quizzes with multiple question types
- 🎯 Real-time quiz-taking interface
- 📊 Score calculation and performance analytics
- 🔐 User authentication and role management (Admin/Student)
- 📚 Question bank management
- 🌐 Responsive UI for desktop and mobile devices

## 🛠️ Tech Stack

- **Frontend:** React.js / HTML / CSS / Bootstrap (or Tailwind)
- **Backend:** Node.js / Express.js / Django / Flask (select one based on your project)
- **Database:** MongoDB / MySQL / PostgreSQL
- **Authentication:** JWT / OAuth
- **Hosting:** (e.g. Vercel, Netlify, Heroku)
